# 巫巫女睡前故事世界观设定文档

## 一、核心角色设定

### 1.1 巫巫女
**基本信息**
- 姓名：巫巫女（第01篇后被称为"心愿女巫"）
- 年龄：年龄不详，像奶奶一样慈祥
- 性格：温柔、智慧、善良，喜欢帮助别人

**外貌特征**
- 头发：乱蓬蓬的紫头发，后期会变得彩色
- 鼻子：尖尖的
- 服饰：穿着彩虹披风（第01篇后开始穿），会轻轻飘动
- 道具：魔杖，洒出亮晶晶的善意粉末
- 居住：森林深处歪歪扭扭的小木屋，屋里摆满了收集快乐光芒的小瓶子

**能力**
- 可以用魔杖洒出善意粉末
- 拥有各种魔法道具（月光坚果、梦境花田、耐心沙漏、思念共鸣瓶、季节调色盘、梦境食谱、勇气小标记等）
- 可以收集"快乐光芒"，存储在小瓶子里

**身份转变**
- 第01篇前：愤怒女巫，收集负面情绪
- 第01篇后：心愿女巫，收集快乐光芒，帮助小动物和孩子们实现善良的愿望

**童年设定**（第13篇）
- 巫巫女小时候也是个小女孩，和莉莉一样大
- 外貌：扎着小辫子，穿着彩虹色的小裙子
- 居住：住在森林里
- 性格：喜欢帮助小动物，和莉莉一样善良
- 记录：有相册记录她长大的样子，相册放在小木屋的书架上

### 1.2 莉莉
**基本信息**
- 姓名：莉莉
- 年龄：6岁
- 性格：勇敢、善良、喜欢帮助别人，好奇心强
- 家庭：和妈妈一起生活，住在森林附近

**特点**
- 喜欢和巫巫女一起帮助森林里的小动物
- 经常主动提出各种想法和计划
- 每次遇到问题都会积极想办法解决
- 当前阶段：幼儿园大班上学期结束，已经学会认识简单汉字、照顾中班小班的弟弟妹妹、轮流和等待，正在准备学习拼音，对数学思维游戏等学习类游戏充满兴趣（第12篇）
- 寒假活动：参加了寒假班，在寒假班结识了新朋友欣欣（第14篇）

### 1.3 莉莉妈妈
**基本信息**
- 姓名：莉莉妈妈
- 性格：温柔、支持女儿，经常和莉莉一起参与冒险

**特点**
- 经常和莉莉一起准备食物、制作点心
- 有时会加班，和莉莉有"思念"的互动（第07篇）

### 1.4 欣欣
**基本信息**
- 姓名：欣欣
- 年龄：和莉莉差不多大
- 性格：安静、喜欢画画、细心、心灵手巧
- 家庭：刚搬到市民广场附近

**外貌特征**
- 发型：梳着丸子头
- 服饰：穿黄色羽绒服
- 配饰：戴毛线帽（冬天）

**特点**
- 喜欢画画，随身携带画本和画笔，背着画板
- 对大自然有兴趣，但不太认识植物
- 心灵手巧，擅长把东西摆得漂漂亮亮
- 在寒假班和莉莉是同学，通过巫巫女成为好朋友（第14篇）
- 学会了观察和记录，能独自去森林找巫巫女（第20篇）
- 成长经历：在莉莉旅行期间，学会独立享受生活，发现属于自己的小秘密（第20篇）；与莉莉一起经历"成长的小雨滴"，学会接纳不完美的自己（第21篇）

### 1.5 游轮旅行中的新朋友（第15-19篇）

**乐乐**
- 性别：男孩
- 年龄：和莉莉差不多大
- 特点：喜欢用望远镜看海豚，对海洋充满好奇

**小雅**
- 性别：女孩
- 年龄：5岁
- 特点：第一次坐游轮，容易走散但很勇敢，和莉莉成为好朋友，约定一起看日出

**豆豆**
- 性别：男孩
- 外貌：戴眼镜
- 特长：会弹电子琴
- 特点：提议组建"船舱里的秘密乐队"

**琪琪**
- 性别：女孩
- 外貌：扎马尾辫
- 特长：会唱歌
- 特点：秘密乐队成员之一

**船长爷爷**
- 外貌：穿蓝色制服，有胡子
- 性格：和蔼可亲
- 特点：邀请莉莉参观驾驶室，讲述驾驶游轮需要勇气

## 三、场景设定

### 3.1 巫巫女的小木屋
- 位置：森林深处
- 外观：歪歪扭扭的
- 内部：摆满了收集快乐光芒的小瓶子，有书架（书架上放着相册等物品）
- 门前：有台阶，巫巫女经常坐在台阶上

### 3.2 森林场景
- 彩虹瀑布
- 月光崖
- 心愿树（第05篇）
- 梦境花田（第04篇）
- 野花坡（第10篇）
- 雨中的小木屋茶会场景：小木屋屋檐下的小木桌和小木凳、雨帘和彩虹、雨中的金银花茶（第11篇）
- 森林小动物们的家（朵朵的树屋、白白的菜地、慢慢的池塘边）

### 3.3 森林外的场景
- 市民广场：雪后的广场、花坛、松树、腊梅、冬青（第14篇）
- 寒假班教室：莉莉和欣欣上课的地方（第14篇）

### 3.4 春节游轮场景（第15-19篇）
- **上海宝山码头**：游轮出发地，农历正月初五（破五）
- **游轮设施**：
  - 甲板：挂红灯笼、贴春联福字、有栏杆和躺椅
  - 驾驶室：各种仪表盘和按钮，大方向盘
  - 自助餐厅：供应食物和饮料
  - 活动室：儿童活动区，可以排练表演
  - 巫巫女的独立小屋：甲板尽头，适合收集心愿
  - 房间阳台：可以看海、看星星
  - 广播室：负责播报寻人信息
- **海上景观**：
  - 大海：蓝得像巨大宝石
  - 海豚岛：远处的绿色小岛，有海豚游玩
  - 海上日出：金色阳光洒在海面
  - 小岛沙滩：可以登陆的小岛，有螃蟹和彩色小鱼
- **旅程信息**：五天四晚（实际4天5晚），春节期间

## 六、故事风格与模式

### 6.1 故事风格
- 温馨治愈
- 情节简单、有画面感
- 适合6岁小女孩聆听
- 每篇故事包含一个温和的教育主题（耐心、思念、合作、陪伴等）

### 6.2 故事模式
1. **开头**：莉莉发现问题或提出想法 → 找到巫巫女 → 巫巫女提供帮助
2. **中间**：大家一起冒险/解决问题 → 温馨互动 → 感悟主题
3. **结尾**：**必须包含甜梦场景**（详见下文）

## 七、结尾甜梦场景设定（核心）

**这是最重要的设定，每篇故事（除了第01篇）都必须遵循！**

### 7.1 甜梦场景的结构

1. **小动物们的甜梦**
   - 分别描述每个出场的动物在梦里的样子
   - 梦境内容要符合动物性格（比如朵朵梦见松子、白白梦见胡萝卜）
   - 用温馨、可爱的语言描述

2. **莉莉的甜梦**
   - 莉莉躺在床上，抱着某样东西（花环、围巾、沙漏模型等）
   - 梦见和巫巫女、小动物们、妈妈一起玩耍
   - 梦境内容要与当天故事的主题呼应

3. **巫巫女的总结**
   - 巫巫女坐在小木屋的台阶上或坐在屋里
   - 看着收集瓶（如果收集了光芒）或静静感受夜晚
   - 说一句温柔的话，点明故事主题
   - 常用句式："善意魔法，陪你做甜甜的梦"、"XXX就是最甜的魔法呀"

4. **最后的氛围描写**
   - 月光、小木屋、香气守护甜梦
   - 用轻柔的语言结束故事

### 7.2 第01篇的特殊性
- 第01篇作为开篇，结尾没有甜梦设定
- 结尾是："好的故事就像一颗种子，在小小的心里慢慢发芽🌱"
- 从第02篇开始，每篇都必须有甜梦场景

### 7.3 甜梦场景示例（来自第10篇）

**小动物们的梦**：
- 朵朵梦见自己躺在鲜花野餐垫上，周围是数不完的松子
- 白白梦见自己编了一个超级大的花环
- 球球梦见背上的刺果都变成了超级大的甜浆果
- 慢慢梦见自己趴在野花坡上，周围开满了慢吞吞的小花
- 跳跳梦见自己坐在荷叶上，在花海里划船

**莉莉的梦**：
- 梦见自己和妈妈、巫巫女，还有朵朵、白白、球球、慢慢、跳跳，在开满鲜花的草地上野餐
- 巫巫女编了一个更大的花环，挂在月亮上，月亮就变成了花环月亮

**巫巫女的总结**：
- 巫巫女坐在小木屋的台阶上，看着月亮升起来
- 轻轻笑起来，感受着这份温柔的安静
- 小声说："春日里的陪伴和欢笑就是最甜的魔法呀。善意魔法，守护着每一个甜甜的梦。"

**最后的氛围描写**：
- 夜晚的森林静悄悄的，月光把野花坡照得亮晶晶的
- 白天的花香还在空气里轻轻飘荡，守护着每一个甜甜的梦

## 十、创作新故事时的注意事项

### 10.1 必须遵守的规则
1. 保持角色设定一致（巫巫女、莉莉、小动物们的性格不能改变）
2. 从第02篇起，每篇结尾必须包含完整的甜梦场景
3. 光芒收集不是必需的，根据主题决定是否收集
4. 语言风格要温馨治愈，适合6岁小女孩
5. 情节简单、有画面感，不要太复杂

### 10.2 可以灵活的元素
1. 光芒收集：可以收集新光芒，也可以不收集
2. 巫巫女的总结语：不拘泥于固定句式，根据主题自然表达
3. 故事主题：可以是日常治愈、成长教育、冒险探索等
4. 小动物出场：根据需要选择哪些动物出场

### 10.3 禁止事项
1. 禁止改变巫巫女的核心设定（外貌、身份、性格）
2. 禁止改变莉莉的年龄和基本性格
3. 禁止跳过甜梦场景（第02篇起）
4. 禁止使用恐怖、暴力的元素
5. 禁止让故事情节过于复杂或逻辑混乱

## 十一、互动分支设计框架（可选）

### 11.1 互动节点设计
在故事的关键节点，可以让用户（小女孩）做选择，影响后续剧情。

**示例**：
- 故事讲到一半："莉莉看到一条岔路，左边是开满鲜花的路，右边是通往月光崖的路。你觉得她该走哪条？"
- 用户选择："月光崖"
- 后续：生成"巫巫女的月光崖探险"的内容

### 11.2 分支类型
1. 路线选择：花海 vs 月光崖 vs 梦境森林
2. 行为选择：帮助小动物 vs 自己冒险
3. 情绪选择：开心的冒险 vs 温馨的日常

### 11.3 注意事项
- 分支不要太多，保持在2-3个选项
- 每个选择都要有明确的后续方向
- 选项的描述要符合6岁孩子的理解能力

## 十二、音频生成规范

### 12.1 配音风格
- 默认音色：温柔女声 - vivi
- 语速：舒缓，适合睡前
- 语气：温柔、亲切、有节奏感

### 12.2 开场规范
- 禁止使用通用的"欢迎收听扣子AI播客"开场
- 开场要符合睡前故事氛围，比如：
  - "夜晚的森林..."
  - "月亮慢慢升起来了..."
  - "春风吹过森林..."

### 12.3 甜梦场景的配音
- 语速更慢，更温柔
- 可以加入背景音乐（摇篮曲、自然白噪音）
- 每个角色的梦描述时要切换语气

### 12.4 音频时长
- 控制在5-10分钟
- 适合睡前聆听
